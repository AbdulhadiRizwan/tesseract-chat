Tesseract

A private, multi-user chat app with direct messaging, file sharing, voice recording, and local AI — no cloud required.

⬇️ Download Tesseract ZIP ~2 MB

Features

💬

Direct Messages

Private 1-on-1 conversations. Full chat history saved locally.

📎

File Sharing

Send photos, videos, and audio files directly in chat.

🎤

Voice Recording

Record and send voice messages with one click.

🔍

User Search

Find any user and start a conversation instantly.

🔒

Private Server

Run your own server. No data leaves your network.

🗄️

Persistent History

All messages stored in SQLite — never lose a conversation.

How to Install

  1. Download & extract the ZIP file above.
  2. Install Python 3.9+ if you don't have it: python3 --version
  3. Install dependencies: pip install -r requirements.txt
  4. Run the app: python3 client.py
  5. Click "Start Server" to host, or connect to a friend's server IP. Register an account and start chatting!

Requirements

🐍 Python 3.9+ ⚡ PySide6 🔌 Flask-SocketIO 🎵 sounddevice 🗄️ SQLite
